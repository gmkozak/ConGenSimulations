# ConGenSimulations

This is a repository of Jupyter Notebooks and code for simulations in a Conservation Genetics course.

The Jupyter notebooks using the package cgsim (Zimmerman & Oyler-McCance 2025)

Zimmerman, S. J., S. J. Oyler-McCance. 2025. cgsim: An R package for simulating conservation relevant outcomes for management actions, (version 1.0.1): U.S. Geological Survey software release, https://doi.org/10.5066/P1BXBEXJ.
